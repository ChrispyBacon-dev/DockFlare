# Benvenuti nella documentazione di DockFlare!

DockFlare è un potente controller ingress self-hosted che semplifica la gestione di Cloudflare Tunnel e Zero Trust. Utilizza le etichette Docker per la configurazione automatica e offre una Web UI solida per definizioni manuali dei servizi e override delle policy.

Questa documentazione fornisce informazioni complete su DockFlare. Che tu sia un nuovo utente o un utente esperto, troverai tutto ciò che devi sapere per ottenere il massimo da DockFlare.
